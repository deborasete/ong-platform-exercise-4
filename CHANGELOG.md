# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-03
### Added
- Implemented **WCAG 2.1 Level AA** accessibility improvements:
  - Keyboard navigation support for all interactive components  
  - Semantic HTML structure for improved screen reader compatibility  
  - Minimum color contrast ratio of **4.5:1** for normal text  
  - Full **screen reader support** using appropriate ARIA attributes  
  - Accessible **high-contrast** and **dark mode** themes
